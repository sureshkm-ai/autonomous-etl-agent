Configuration Reference

All configuration is managed through pydantic-settings (src/etl_agent/core/config.py). Values are read from environment variables or from a .env file in the project root. In production, secrets are injected by ECS from AWS Secrets Manager.

from etl_agent.core.config import get_settings
settings = get_settings()    # returns a cached Settings instance

Variable

Default

Description

SQS_QUEUE_URL

(empty)

Full SQS URL. When set, enables ECS Fargate mode (messages are enqueued vs inline).
